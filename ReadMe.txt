‘Slimy Doodly’ is 2D side-view puzzle platformer game in which the player uses his wave gun to push slime or to activate slime’s ability. Slimy Doodly is a game where the player solves puzzles and escapes using wave guns and slime. The hint for each puzzle is just to remember and use the characteristics of the given elements, so the player's judgment and memory is required a little bit.

How To Play
- Game's goal is simple, solve the puzzle and get to the next stage.
  Use wave to jump higher or move slime to solve puzzle.
  Trigger tiles enable when object is on it, and have its' unique behavior.
  You could also challange to the hard mode. At hard mode, restart the game will make you get to the
  First Level, and it records whole time to beat the all levels.
  Good luck!


Controls
- In menu
	W, S - move cursor
	Spacebar - Select Menu

- In game
	A D - Move left, right
	W - Jump
	Mouse Left Button - Shoot Wave
	Mouse Right Button - Zoom out
	Mouse Move - Move aim
